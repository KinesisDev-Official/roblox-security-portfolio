CriticalCritique

Backdoor ScriptsScripts Backdoor

Hidden remote events that allow attackers to run arbitrary code in your game at any time, even after you publish updates.

Événements distants cachés permettant à des attaquants d'exécuter du code arbitraire dans votre jeu, même après des mises à jour.

CriticalCritique

Fake Admin ScriptsFaux Scripts Admin

Malicious admin systems that look legitimate but secretly give hackers god-mode access to your entire game.

Systèmes d'admin malveillants qui semblent légitimes mais donnent secrètement un accès total aux hackers.

HighÉlevé

Popup InjectorsInjecteurs Popup

Scripts that display scam links, fake giveaways or Robux phishing in your game's UI, targeting your players.

Scripts qui affichent des liens frauduleux, des faux cadeaux ou du phishing Robux dans votre jeu.

HighÉlevé

DataStore ExploitsExploits DataStore

Unprotected remote functions that allow players to read or corrupt other players' data and game economy.

Fonctions distantes non protégées permettant aux joueurs de lire ou corrompre les données d'autres joueurs.

MediumMoyen

Script Lag / LoopsLag / Boucles Infinies

Unoptimized or leftover scripts causing unnecessary server load, FPS drops, and poor player experience.

Scripts non optimisés causant une charge serveur inutile, des chutes de FPS et une mauvaise expérience joueur.

MediumMoyen

Remote VulnerabilitiesVulnérabilités Remote

RemoteEvents and RemoteFunctions with no server-side validation, exploitable by any cheating client.

RemoteEvents et RemoteFunctions sans validation côté serveur, exploitables par n'importe quel client tricheur.
